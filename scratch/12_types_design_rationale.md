# 設計根拠：人間だからこそ起こりうる「愛すべき脳内バグ」12タイプ

本ドキュメントは、LogiFitの診断システムを「12タイプ認知バグ診断（アたまのレントゲン）」へピボットするにあたり、その分類の心理学的エビデンスおよび数学的整合性を記録したものです。

---

## 1. なぜ「12タイプ」なのか（分類の数学的・論理的整合性）

4つの思考特性（L: Logical / C: Critical / R: Radical / E: Emotional）を「思考のギア」とし、これらが極端に働いたり、脳内で衝突したりした際に発生する「人間味あふれる思考の歪み（バグ）」を分類すると、数学的かつ論理的に**必然の12パターン**が導出されます。

```
【内訳】
- 相反する2つのギアが脳内で衝突している「脳内ジレンマ（葛藤）型」: 6タイプ (4C2)
- 特定の1つのギアが極端に過剰暴走している「単一暴走型」: 4タイプ (4C1)
- 4つのギアが均等に調和・非調和している「バランス型」: 2タイプ
---------------------------------------------------------------------
合計: 12タイプ
```

---

## 2. 心理学的エビデンスとの接続

本分類は、以下の確立された心理学・行動経済学・精神医学の理論と接続しています。

1. **認知行動療法（CBT）における「認知の歪み（Cognitive Distortions）」**
   - アーロン・ベックやデビッド・バーンズらが実証した「現実を不正確に捉えてしまう思考の癖」。
2. **二重過程理論（System 1 & System 2）**
   - ダニエル・カーネマンらが提唱した、脳内の「直感的思考（システム1 / Eなど）」と「論理的思考（システム2 / L・Cなど）」が対立し、バイアスを生み出す仕組み。
3. **防衛機制（Defense Mechanisms）**
   - 不安や心の傷から自分を守るため、脳が自動的に論理化（知性化）や責任転嫁を行う仕組み。

「バグ」とは単なる欠陥ではなく、**「人間だからこそ、自分自身を守ろうとしたり、一生懸命考えようとした結果として発生してしまう、愛すべき機能不全（副作用）」**として定義されます。

## 3. 12タイプ・脳内バグ詳細マトリクス

### I. 脳内ジレンマ（葛藤）型（6タイプ）

| ギア衝突 | タイプ名 | 発生する脳内バグ（心理学） | あるあるシーン / 認知の歪み |
| :--- | :--- | :--- | :--- |
| **L（論理）<br>×<br>E（感情）** | **認知的不協和バグ** | **自己防衛的合理化（言い訳製造）** | 頭では「絶対に自分が悪い」と分かっているのに、心が傷つかないように「でも〇〇だったから仕方ない」という屁理屈を高速で組み立てる。 |
| **C（批判）<br>×<br>E（感情）** | **マインドリーディングバグ** | **深読み過呼吸（過剰警戒）** | 他人の感情に超敏感（E）でありながら、「自分は悪く思われているのでは」と批判的に裏読み（C）するため、相手の些細な表情で自爆する。 |
| **L（論理）<br>×<br>C（批判）** | **分析麻痺バグ** | **決定フリーズ（立ち往生）** | 完璧な計画（L）を立てようとするたびに、自分の批判精神（C）が「リスクはどうする？」と突っ込む。粗探しが終わりを迎えずにフリーズする。 |
| **R（根本）<br>×<br>E（感情）** | **希望的観測バグ** | **お花畑スキップ（現実逃避）** | 「人は分かり合える」という理想（R）と「ラクになりたい」感情（E）が結びつき、現実的なエビデンスやリスク（L/C）を無視して暴走する。 |
| **R（根本）<br>×<br>C（批判）** | **シニカルバグ** | **斜に構え小言（冷笑主義）** | 世の中の前提（R）と欠陥（C）ばかりが見えるため、他人が楽しそうにしていることや、社会の習慣に対して「どうせ意味ない」と冷笑的になる。 |
| **L（論理）<br>×<br>R（根本）** | **机上の空論バグ** | **成層圏浮遊（生活感フリーズ）** | 頭の中のシステムやビジョン（L/R）を美しくすることに夢中で、「ゴミ出し」「部屋の掃除」「予定調整」といった現実手続きの脳内メモリが蒸発する。 |

### II. 単一暴走型（4タイプ）

特定のギアが他のギアのブレーキを無視して過剰に回転してしまっている状態です。

| 暴走ギア | タイプ名 | 発生する脳内バグ（心理学） | 特徴と副作用 |
| :--- | :--- | :--- | :--- |
| **L（論理）極** | **アレキシサイミアバグ** | **置物ロボット化（感情遮断）** | 論理だけで動きすぎ、感情（E）をノイズ扱い。パートナーが泣いて愚痴を言っていても、「泣いても解決しないよ」とファクトで殴りに行く。 |
| **C（批判）極** | **敵意帰属バイアスバグ** | **クレーマーレーダー（全方位警戒）** | 批判が暴走し、世の中のすべての物事に悪意やリスクが潜んでいると錯覚。他人の好意やアドバイスすら「裏がある」と受け取り拒否する。 |
| **R（根本）極** | **抽象無限ループバグ** | **そもそも迷子（目的論中毒）** | 「そもそも」を追求しすぎて、今日のミーティングで決めるべきアジェンダから、宇宙の誕生レベルの根本概念に話を引き戻して進行を止める。 |
| **E（感情）極** | **感情的決め付けバグ** | **暴走ジェットコースター** | 「私がこう感じるから絶対に正しい！」とノリと一時的なパッションで走り、後から「なんであんなことしたんだろう…」と激しく後悔する。 |

## 4. 人と比べたくなる共有（バイラル）の仕掛け

この「葛藤と愛すべきバグ」モデルは、以下のようなSNSシェア行動を強力に促進します。

* **自慢ではなく「自虐」だからシェアしやすい**: 
  「私は完璧なロジカルです」という自慢ではなく、「私の脳内ではLとEがいつもプロレスしていて、結果として言い訳製造バグが発生してます笑」という表現になるため、心理的ハードルが低く共感を呼びます。
* **他者スキャナーとの相乗効果**:
  「上司をスキャンしたら『冷笑シニカルバグ』で、私は『深読み過呼吸バグ』だった。相性摩擦係数85%になるわけだ」と、納得感とともに会話が生まれ、対比シェアの動機を刺激します。

## 5. プロジェクト独自アルゴリズムの定義

本診断モデルで採用している判定ロジックは、既存のタイプ診断ツール（MBTIやエニアグラム）の焼き直しではなく、LogiFitが独自に設計・定式化したオリジナルのアルゴリズムです。

### アルゴリズム正式名称
**「LCREギア摩擦マッピング・アルゴリズム (LCRE Gear Friction Mapping Algorithm)」**
（開発コードネーム：*Brain Dilemma Scanning Engine (BDSE)*）

### 数学的・論理的モデル

本アルゴリズムは、ユーザーの診断回答から得られた4つの思考次元ベクトル $\vec{V} = (L, C, R, E)$ （各成分の範囲: $0 \le x \le 105$ ）を入力とし、以下の3段階の判定プロセスを経て、人間の脳内の「摩擦・葛藤・バグ」を抽出します。

#### 【第1段階：多重均衡（バランサー）判定】
4軸の最大値と最小値の差（分散度）が極めて小さい状態を検知します。
$$\text{Range} = \max(L, C, R, E) - \min(L, C, R, E)$$
*   **$\text{Range} \le 15$ の場合**: すべてのギアが等しく噛み合う「バランス型」と判定。
    *   $\text{Average}(L, C, R, E) \ge 20$ ➡ **① 深思のハイエロファント（知恵の神官）**
    *   $\text{Average}(L, C, R, E) < 20$ ➡ **② 野生のトリックスター（風の妖精）**

#### 【第2段階：単一過剰（オーバードライブ）判定】
特定の1つのギアが、ブレーキが壊れたように突出している状態を検知します。
1位のスコアを $M_1$、2位のスコアを $M_2$ とします。
*   **$M_1 - M_2 \ge 30$ の場合**: 単一ギアの「暴走型」と判定。最もスコアが高いギアに応じて決定。
    *   $L$ 突出 ➡ **③ アレキシサイミアバグ（置物ロボット化）**
    *   $C$ 突出 ➡ **④ 敵意帰属バイアスバグ（クレーマーレーダー）**
    *   $R$ 突出 ➡ **⑤ そもそも迷子バグ（目的論中毒）**
    *   $E$ 突出 ➡ **⑥ 感情的決め付けバグ（暴走ジェットコースター）**

#### 【第3段階：二重葛藤（ジレンマ）判定】
突出した上位2つのギアが拮抗し、脳内で激しい摩擦（ジレンマ）を起こしている状態を検知します。
*   **$M_1 - M_2 < 30$ の場合**: 2ギアの「葛藤型」と判定。上位2つの組み合わせによって決定。
    *   $L \times E$ ➡ **⑦ 認知的不協和バグ（自己防衛的合理化 / 言い訳製造）**
    *   $C \times E$ ➡ **⑧ マインドリーディングバグ（深読み過呼吸 / 過剰警戒）**
    *   $L \times C$ ➡ **⑨ 分析麻痺バグ（決定フリーズ / 立ち往生）**
    *   $R \times E$ ➡ **⑩ 希望的観測バグ（お花畑スキップ / 現実逃避）**
    *   $R \times C$ ➡ **⑪ シニカルバグ（斜に構え小言 / 冷笑主義）**
    *   $L \times R$ ➡ **⑫ 机上の空論バグ（成層圏浮遊 / 生活感フリーズ）**

この階層的ベクトルマッピングモデルにより、ユーザーが意識していない「内面のジレンマや思考ギアの摩擦状態」を科学的・論理的にレントゲン撮影することが可能となります。
